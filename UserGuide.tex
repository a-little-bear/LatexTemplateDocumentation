\section{Introduction}

\noindent
The DocumentClass \textit{hw.cls} aims to provide a simple and easy template for writing math notes on latex. The link to hw.cls, Example.tex, and Template.tex can be found at \url{https://github.com/a-little-bear/Latex-Template}.\\

\noindent
Here is the template of a new tex file (note that all 6 \verb=\def= and the cover command are optional): 
\begin{latexcode}
    \documentclass[12pt, brown, sepia, 0.5in]{hw}

    % \def\course{}

    % \def\email{}

    % \def\logo{}

\newpage\subsection{Symbol Abbreviations}\hfill


\subsubsection{Greek Letters}

All letters have been shortened into the first 2 letters (or 1 if the total length is 1). For example,
\begin{verbatim}
    \al -> \alpha
    \Al -> \Alpha
    \be -> \beta
    \ph -> \phi
    \om -> \omega
    \Om -> \Omega
\end{verbatim}

The variable versions are also included, for example,
\begin{verbatim}
    vep -> \varepsilon
    vph -> \varphi
    vOm -> \varOmega
\end{verbatim}



\subsubsection{Other symbols and Operations}
\begin{verbatim}
  \C -> \mathbb{C} == complex
  \R -> \mathbb{R} == reals
  \Q -> \mathbb{Q} == rationals
  \Z -> \mathbb{Z} == integers
  \N -> \mathbb{N} == naturals 
  \F -> \mathbb{F} == field

  \p -> \partial

  \? -> \stackrel{?}{=} == question mark on equal sign
  \ra -> \rightarrow == rightarrow (single line)
  \Ra -> \Rightarrow == Rightarrow (double lines)
  \is -> \equiv == equivalent (triple lines)
  \sse -> \subseteq
  \injective \surjective \bijective

  \arr = angle brackets
  \bra = parenthesis ()
  \sqrbra = square brackets []
  \curbra = curly brackets {}
  \abs = absolute value | |
  \norm = double absolute || ||
  \ceil = ceiling + () + ceiling
  \floor = floor + () + floor
  \near = floor + () + ceiling

  \func[3]{#1: #2 \rightarrow #3} == function (name, domain, codomain)
  \Pset{#} -> \mathcal{P}(#) == power set
  \Relate{#}{##} -> #\mathcal{R}## == relation
  \GF[1][2]{\bb{F}_{#1}} == Galois field, default #1 = 2
  \modulo[1][n]{\Z/#1\Z} == modulo, default #1 = n
  
  \P -> \mathbb{P} == primes
  \nil -> \varnothing == empty set
  \O -> \mathcal{O} == big O 
  \relate -> \mathcal{R} == relate (relation)

  \Dar[1][1] = scaled Downarrow
  \dar[1][1] = scaled downarrow
  \Uar[1][1] = scaled Uparrow
  \uar[1][1] = scaled uparrow
\end{verbatim}

\subsection{Image and Listing}

\subsubsection{Image}

Floating and centered figure with fixed ratio:
\begin{verbatim}
    \fig[
        width=0.5\textwidth,
        height=0.5\textheight,
        label={},
        caption={},
        Optionals
    ]{path}
\end{verbatim}
Only path is required, optional keys are also available.

\subsubsection{Listing}

Listings need to be accessed through the begin and end method:
\begin{verbatim}
    \begin{code}[language=\defaultlanguage]
    ...
    \end{code}

    \begin{latexcode}[language=[LaTeX]Tex]
    ...
    \end{latexcode}
\end{verbatim}

\subsection{Alg}

This is a package designed for pseudocode. For sample usage please refer to Example.tex

\verb=\usepackage[noend,noline]{clsfiles/alg}=

noend parameter removes the ending keywards, noline removes the line numbers.

To use the algorithm / aogirithmic environment: \verb=\newalg[]{}= where the optional parameter is the title of the algorithm.

\section{Known Bugs / Improvements}

\begin{enumerate}
    \item Nest chapter, exercise, unit together are unlickly to work.
    \item To improve readability, the environments should try to not be nested, especially during page breaking. 
    \item The number counters cannot align with section numbers (counters are not well defined).
    \item The color box sometimes touches the footer.
    \item Make sure use \verb=\np= or \verb=\newpage= for new exercise/unit/chapter so that the page splitting functions properly (due to nested tcolorbox force breakable bug).
    \item Allow more line spacing for alg package.
\end{enumerate}
